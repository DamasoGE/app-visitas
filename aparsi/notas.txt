DEJANDO A UN LADO EXPO GO -> EXPO DEV
-------------------------
# 1. Instala expo-dev-client
npx expo install expo-dev-client

# 2. Prebuild
npx expo prebuild

# 3. Instala la app personalizada en tu emulador
npx expo run:android

# 4. Inicia el servidor de desarrollo con recarga
npx expo start --dev-client

# Por si hay algún problema de dependencias
npx expo install --check
